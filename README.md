# lab

Monorepo for experimental projects.

- [capsa](capsa/) — Lightweight VMs with network sandboxing, powered by libkrun.
- [fittings](fittings/) — A Rust library for building JSON-RPC 2.0 / MCP services.
- [lockin](lockin/) — Cross-platform process sandboxing (Linux via syd, macOS via sandbox-exec).
- [outpost](outpost/) — Host-allowlist networking policy library and loopback HTTP CONNECT proxy, shared by capsa and lockin.
- [scope](scope/) — Non-interactive CLI web browser for AI agents (search, read URLs as markdown).
- [tempo](tempo/) — CLI for benchmarking LLM inference throughput over OpenAI-compatible streaming APIs.
